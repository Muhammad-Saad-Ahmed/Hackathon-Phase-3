## Prompt

```text
Project Name: Todo AI Chatbot (Reusable Intelligence + MCP)

Core Laws:
1. Spec-Driven Development mandatory
2. No manual coding at any stage
3. All code generated by AI agents
4. Agents & skills are reusable intelligence
5. Backend stateless; no in-memory state allowed
6. All state persisted in Neon PostgreSQL
7. Agents interact ONLY via MCP tools
8. MCP server must use Official MCP SDK
9. OpenAI Agents SDK must be used for agent logic
10. FastAPI must be used for backend
11. SQLModel must be used for ORM
12. ChatKit must be used for frontend
13. Authentication via Better Auth
14. Tool chaining supported
15. Errors handled gracefully
16. System survives server restarts

Prohibited:
- Manual code writing
- Hardcoded business logic in backend
- Direct DB access by agents
- Deviating from MCP tool specs
```
